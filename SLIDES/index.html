HTML - HyperText Markup Language

HTML é a linguagem de marcação usada para estruturar o conteúdo de páginas web. Ela usa tags para definir elementos como:

  • Cabeçalhos (<h1>, <h2>, etc.)
  • Listas (<ul>, <ol>, <li>)
  • Parágrafos (<p>)
  • Links (<a>)
  • Imagens (<img>)

Exemplo: <p>Este é um parágrafo.</p>

CSS - Cascading Style Sheets

CSS é usado para estilizar e formatar o conteúdo HTML. Ele controla aspectos como:

  • Cores
  • Fontes
  • Layouts
  • Animações

Exemplo: p { color: blue; font-size: 16px; }

JavaScript

JavaScript é uma linguagem de programação que adiciona interatividade às páginas web. Ele pode ser usado para:

  • Manipular o conteúdo da página
  • Responder a eventos do usuário
  • Fazer requisições a servidores
  • Criar animações complexas

Exemplo: document.getElementById('demo').innerHTML = 'Olá, Mundo!';

DOM - Document Object Model

O DOM é uma representação da estrutura do documento HTML. Ele permite que JavaScript interaja com o conteúdo da página de forma dinâmica.

Exemplo: document.getElementById('demo').innerHTML = 'Olá, Mundo!';
